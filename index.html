Learn how to use CSS Grid to build out a full, actual webpage. This series explores using the CSS grid in the real world, building out a site from scratch, including using fallbacks for browsers which don't support css grid. In this video, I introduce the project and then write the markup (HTML) for the page. While this series focuses on how we can build out a full page with the CSS Grid, I'll also be using Flexbox for my layout, and at the end, we'll wrap everything up with some JavaScript to add functionality to the site.

I'm seeing this series as a nice tutorial on how to use the CSS Grid in more real-world scenario than what I've covered so far. Each section has its own challenges and will let us put different tools from the grid to use. It should be a fun one guys! If you'd like to follow along with me, you've got a few options.
